A bespoke one-pager prepared for Invelon Technologies (eMerge Americas 2026, Booth 338). Visual system is drawn entirely from Invelon's own brand tokens: Rajdhani headings, Roboto body, crimson accents (#890631, #B4003C, #D50037) on white with a neutral-dark CTA band, Prismic-style grid chrome, and a single crimson accent column anchoring the hero. The page carries the pitch body from `fit-analysis.md` section 7 verbatim, with the 60/30/10 stack view rendered as a three-cell frame and the Correlation One engagement rendered as a tight metrics column. The CTA routes to Matt Creamer (CRO) for a 30-minute methodology conversation. Content is prospect-facing only: no engagement fees, no retainer or revenue-share figures, and the Eduba wordmark appears solely in the footer.
